11.3.86 freeze/2

Synopsis

freeze(+Flag, +Goal)

Blocks Goal until Flag is bound.

Arguments

Flag

term

:Goal

callable, must be nonvar

Description

Defined as if by: 

freeze(X, Goal) :- when(nonvar(X), Goal).

or 

:- block freeze(-, ?).
freeze(_, Goal) :- Goal.

Backtracking

Depends on Goal.

Exceptions

Call errors (see ref-sem-exc).

See Also

ref-sem-sec.
